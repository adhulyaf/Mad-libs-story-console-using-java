
# Mad Libs story Console Game using java 

A fun, interactive console-based game written in Java where the user provides words and the program generates a humorous story.

## Features

* Accepts user input for nouns, verbs, adjectives, names, animals, and places.
* Uses basic Java input handling with `Scanner`.
* Demonstrates string concatenation and simple story formatting.

## How to Run

1. Make sure you have **Java** installed.
2. Clone or download this project.
3. Open the project folder in any Java IDE (e.g., IntelliJ, Eclipse, VS Code with Java extensions) or compile using terminal.

### Compile and Run via Terminal

java Main.java
